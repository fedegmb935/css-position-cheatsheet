CSS Position Property Reference

Position Values

Value Name Description
static Static The default value, the element is positioned according to the normal flow of the document.
relative Relative The element can be moved relative to its default position in the document flow using the top, right, bottom and left properties.
absolute Absolute The element is removed from the normal flow of the document and is positioned relative to its nearest positioned ancestor.
fixed Fixed The element is positioned relative to the viewport, meaning it stays in the same place even when the page is scrolled.
sticky Sticky The element is positioned relative until the specified offset is reached, then it is position fixed.

Offset Properties

Offset Name Description
top Top Moves the element from the top of the page.
bottom Bottom Moves the element from the bottom of the page.
left Left Moves the element from the left of the page.
right Right Moves the element from the right of the page.
